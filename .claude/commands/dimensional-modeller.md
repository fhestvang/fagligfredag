# Dimensional Modeller Command

This command invokes the dimensional-modeling skill for explicit dimensional modeling tasks.

## Usage
Use `/dimensional-modeller` when you want to explicitly engage dimensional modeling expertise.

## Note
The dimensional-modeling skill auto-activates when discussing:
- Star schema, facts, dimensions
- Grain, surrogate keys, SCDs
- Bus matrix, data warehouse design
- Kimball patterns, denormalization

## Invoke Skill
Load the dimensional-modeling skill and apply its methodology to the current task.

See `.claude/skills/dimensional-modeling/SKILL.md` for full documentation.
